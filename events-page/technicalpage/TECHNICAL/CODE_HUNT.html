EVENT DATE:-
EVENT VENUE:-ANNEX BUILDING,PARKING LOT.
EVENT TIME:- 10:30 – 14:45



The event is inclusive of, people’s riddle cracking and coding skills, which are essential for a programmer or an IT specialist. The event consists of 3 rounds which are based on tech riddles, syntactical and algorithmic riddles and the final round will be based on pure coding the questions will be on data structures and file handling.

Round 1: There will be color coded qr codes which will be linked to teams. The qr codes will open riddles. The location and answer to the next qr code will be in the result. There will be 3 such qr codes.Only after breaking the first riddle and the 2nd riddle and submitting their answers, the next riddles will open. Making sure that no one will get free riddles just by seeing people run towards the riddles.

Round 2: This round will have 2/3 riddles both will be CS based and the same procedure for opening the riddles will apply in this round. The difficulty level will be the same for everyone but it will be spread over the 2 or 3 riddles aggregating to the same difficulty level.

Round 3: This will have pure coding. Where the topmost team from the previous round will get a free choice of questions. These will be based on data structures, file and string handling & mathematical computation
